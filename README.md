# Blazor5Validation
A Blazor Wasm App with the following:
- FluentValidation
- Blazored.FluentValidation
- ApiControllers
- Client Side / Server Side Validation
- CQRS/Mediatr
- Feature Slices

The goal of this is to learn validation techniques split between client side and server side.
